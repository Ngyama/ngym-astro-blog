这是永山鹿夏的个人博客，记录学习、感受与路过风景。

## 关于

- 作者：永山鹿夏
- 内容：学习笔记、随笔、圣地巡礼记录、项目开发日志

## 技术栈

本博客基于以下技术构建：

- [Astro](https://astro.build/) - 静态站点生成器
- [Astro Theme Pure](https://github.com/cworld1/astro-theme-pure) - 博客主题
- [UnoCSS](https://unocss.dev/) - CSS 框架

## 本地开发

```shell
# 安装依赖
bun install

# 启动开发服务器
bun dev

# 构建项目
bun run build
```
